## Directory Structure

```bash
.
├── api         # Kubernetes CRD base types
├── bin         # Built executable binaries
├── chart       # The UI helm chart
├── client      # All client code and built code
├── cmd         # Go main-package code
├── controllers # Controllers for defined CRDs
├── docs        # Markdown documentation
├── e2e         # End-to-end tests
├── hack        # Files used in local development
└── internal    # Internal Go packages
```
